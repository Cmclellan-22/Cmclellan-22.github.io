Final Project

This program is my Final Project for CSC 106. It is a game inspired from the dino game where you jump over obsticles. The code was inspired by the frogger game we made in class.

The game involves jumping over trees to gain score. You jump over a tree to gain score and lose score if you hit a tree. If you get 25 score you win.

The game features the bimojis of my partner Aurthur Souza and me Christian McLellan. Good luck and avoid those trees!
